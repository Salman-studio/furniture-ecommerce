
#  Furniture E-commerce Management System

A complete Furniture E-commerce web application built to manage products, categories, orders, customers, and admin operations.  
This project includes a dynamic front-end, fully functional admin panel, and MySQL database integration.

---

##  Features

### Customer Side
- Browse products by category
- Product details page
- Add to cart functionality
- User registration & login
- Checkout process
- Order tracking

###  Admin Panel
- Admin login system
- Add / Edit / Delete products
- Manage categories & subcategories
- Manage orders (Pending / Completed / Cancelled)
- Customer management
- Dashboard with analytics (Total orders, revenue, status count)

---

##  Tech Stack

| Layer | Technology |
|------|------------|
| **Frontend** | HTML, CSS, JavaScript, Bootstrap |
| **Backend** | PHP (Core PHP) |
| **Database** | MySQL |
| **Server** | XAMPP / Apache |



1.How to Run Locally

Clone the repository:
git clone https://github.com/Salman-studio/furniture-ecommerce.git

2.Move the folder to htdocs (XAMPP).

3.Import the database file:

Open phpMyAdmin

Create a new database (example: furniture_db)

Import the .sql file from the project

4.Update DB credentials in:
includes/db_connection.php

5.Run the project:
http://localhost/furniture-ecommerce/
